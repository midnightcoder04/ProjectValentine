# CASE NO. 2024-DATE: MOTION TO PROCEED

A sophisticated, legally-themed single-page website for a date invitation.

## Features

- **Professional Legal Aesthetic**: Navy blue, parchment white, and muted gold color scheme
- **Typography**: Playfair Display serif for headings, Lato sans-serif for body text
- **Interactive Pen-Flicking Animation**: SVG pen that spins when clicked to unlock the RSVP buttons
- **Three Response Options**:
  - MOTION GRANTED: Plays gavel sound effect and generates a calendar invite
  - ADJOURN FOR DELIBERATION: Opens a text area for questions/discussion
  - MOTION DENIED: Shows a respectful dismissal message
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices
- **Smooth Scrolling**: Elegant navigation between sections

## Sections

1. **The Preamble (Hero)**: Title and case introduction
2. **Statement of Facts**: The date pitch
3. **Compliance & Ethics Declaration**: Humorous reference to judicial ethics
4. **Non-Disclosure Agreement**: Confidentiality clause about the date
5. **Verification of Authority**: Pen-flicking skill check to unlock RSVP
6. **The Verdict**: Three-option response buttons

## How to Use

1. Open `index.html` in a web browser
2. Customize the placeholder text:
   - Replace `[Her Name]` with the recipient's name
   - Replace `[Food Type]` and `[Activity]` with your planned date details
3. Share the website link

## Files

- `index.html` - Main HTML structure
- `styles.css` - All styling and responsive design
- `script.js` - Interactive features and animations

## Technical Details

- No external dependencies (other than Google Fonts)
- Pure vanilla JavaScript
- CSS animations for smooth interactions
- Web Audio API for gavel sound effect
- ICS file generation for calendar invites
